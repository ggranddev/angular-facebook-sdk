# Angular2 Facebook SDK

**NOTE**: This is work under progress. Watch this repo to be updated.

## Installation
Via NPM:
```
npm install ng2-facebook-sdk
```

Via GIT:
```
git clone https://github.com/ihadeed/ng2-facebook-sdk.git
```

Via Bower:
```
bower install ng2-facebook-sdk
```
